снови HTML
Щоб швидко створювати HTML, познайомся з Emmet.

Вебсторінка — це документ, який відображається у веббраузері. Вона може містити різноманітний контент, такий як текст, зображення, 
відео та посилання, і створюється за допомогою мов програмування:

HTML (Hypertext Markup Language) визначає структуру вебсторінок за допомогою HTML-елементів;
CSS (Cascading Style Sheets) визначає, як будуть виглядати HTML-елементи;
JavaScript визначає всі взаємодії з вебсторінкою.
У цьому уроці ти дізнаєшся про основні елементи HTML.

Теги та атрибути
Вебсторінка HTML складається з елементів. Кожен елемент має відкриваючий тег (слово, обгорнуте в <tag>), закриваючий тег (те саме слово, 
обгорнуте в </tag>), та деякий вміст (все між відкриваючим і закриваючим тегами). Деякі елементи не мають вмісту (image, input тощо), 
тому вони не мають закриваючих тегів.

Нижче наведено приклад тегу <p>, який створює абзац на сторінці:

<p>This is a paragraph.</p>

У HTML атрибут — це модифікатор, який надає додаткову інформацію про елемент. Він може впливати на поведінку, зовнішній вигляд чи 
функціональність елемента.

HTML-атрибути додаються до відкриваючого тегу елемента і складаються з пари ім'я-значення. Назва атрибута завжди передує знаку рівності =, а 
значення атрибута обгортається в подвійні лапки.

Наприклад, додати зображення на сторінку можна за допомогою тегу <img> з атрибутами src (адреса зображення) та 
alt (короткий опис зображення):

<img src="example.jpg" alt="Example image">

Зверни увагу: зображення мають лише відкриваючий тег <img>.

Структура HTML-сторінки
Нижче наведена базова структура HTML-сторінки:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mate academy</title>
  </head>
  <body>
    
  </body>
</html>

Де:

<!DOCTYPE html> визначає тип документа та версію HTML та повинен завжди бути першим на сторінці;
<html> є кореневим елементом, де можна встановити мову сторінки. Він містить усі інші елементи;
<head> містить метаінформацію, таку як заголовок сторінки та посилання на зовнішні ресурси;
<meta> надає метадані, наприклад, кодування або параметри вікна відображення (ми обговоримо це пізніше);
<title> встановлює заголовок вкладки браузера;
<body> містить видимий контент сторінки.
Зазвичай HTML-сторінка містить багато інших елементів, залежно від конкретних потреб.

Фавікон
Фавікон (favicon) — це невелике зображення (іконка), яке відображається відразу поруч із назвою сторінки в закладках браузера. Щоб додати 
фавікон, використай тег <link> всередині тегу <head>:

<link rel="icon" href="logo.png">

Атрибут rel вказує, що це фавікон, а href встановлює адресу зображення.

Заголовки
В HTML існує шість рівнів заголовків, від <h1> до <h6>, де <h1> є найважливішим, а <h6> є найменш важливим:

<h1>Heading Level 1</h1>
<h2>Heading Level 2</h2>
<h3>Heading Level 3</h3>
<h4>Heading Level 4</h4>
<h5>Heading Level 5</h5>
<h6>Heading Level 6</h6>

Зверни увагу: На сторінці має бути лише один основний заголовок <h1>. Кожен розділ сторінки може містити свій заголовок <h2>, а 
підрозділ — підзаголовок <h3>, і так далі. Не варто пропускати рівні чи починати ієрархію не з першого рівня. Це поліпшує доступність 
сторінки для користувачів, які використовують програми читання екрана для навігації сторінкою.

Посилання
Посилання можна додати за допомогою тегу <a> з атрибутом href, де вказати URL цільової сторінки або ресурсу:

<a href="https://www.google.com/">Google</a>
<a href="index.html">Home</a>

За замовчуванням, коли ти натискаєш на посилання, цільова сторінка завантажується в тій самій вкладці браузера. Щоб відкрити сторінку в 
новій вкладці, використай атрибут target="_blank":

<a href="https://www.google.com/" target="_blank">
  Google
</a>

Форматування тексту
Найпопулярніші теги для форматування тексту:

<br> додає новий рядок;
<strong> підкреслює, що текст є важливим;
<em> робить наголос на тексті;
<span> дозволяє стилізувати частину тексту.
<p>This is a text paragraph.</p>
<p>This is a text paragraph <br>with line break.</p>
<p>This is a paragraph with <strong>bold text</strong>.</p>
<p>This is a paragraph with <em>italic text</em>.</p>
<p>This is a paragraph with some <span style="color: red;">red</span> text.</p>

Списки
У HTML є два типи списків: маркований та нумерований.

Для створення маркованого списку використовуй тег <ul>, а елементи додавай за допомогою тегу <li>:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Нумерований список можна додати за допомогою тегу <ol>:

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

Списки можна вкладати один до одного:

<ol>
  <li>Item 1
    <ul>
      <li>Item 1-1</li>
      <li>Item 1-2</li>
    </ul>
  </li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

Коментарі
Коментарі в HTML — це текст обгорнутий у <!-- та -->, який ніяк не впливає на зовнішній вигляд сторінки, а лише допомагає програмістам краще 
розуміти логіку програми:

<!-- This is a comment. -->
<!-- <p>This paragraph is not shown on the page</p> -->
